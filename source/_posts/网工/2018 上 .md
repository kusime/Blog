# 传输延时

- 公式就是

  - 传输延时 + 传播延时
  - 这里需要注意
  - `1 MB/s == 1 x 10^6 b/s`
  - `1 Byte = 8 b`
  - `1 x 10^6 us = 1 s`

- 统一的把单位换成 /s 然后再 x 1 x 10^6
