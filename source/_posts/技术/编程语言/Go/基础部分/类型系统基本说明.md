# go 类型系统基本说明

- 类型系统是 ☞ 一个语言的类型体系结构(int , float 等。。)
- 类型系统描述的是在一个语言中各个类型是如何被关联的

# go 静态编译形式的语言

- 因为 go 是静态编译形式的语言，所以需要在编译的时候知道程序里面的值的类型
- 这样的好处有可以让编译器确保程序可以合理的使用数值，有助于减少潜在的内存数值和 bug

# 一些特别的地方

- 有些内部的类型和编码体系的体系结构有关系
- 比如一个 int 类型的数据，可能有些机器就是 64bit 有些就是 32bit
